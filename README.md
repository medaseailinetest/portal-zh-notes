# portal-zh-notes

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
所有页面均为静态内容，不依赖后端服务，也不针对任何特定域名或网站。

## 项目简介

本仓库主要用于存放零散的、独立用途的 HTML 文件。  
这些页面可能包含笔记、工具界面、展示页或临时信息页，彼此之间无强关联。  
每个页面均为自包含的 HTML 文件，可直接通过浏览器打开或部署到静态托管服务。

## 目录结构

```
.
├── README.md              # 本文件
├── index.html             # 可选：仓库根目录下的索引页
└── pages/                 # 存放所有独立 HTML 页面
    ├── page-a.html
    ├── page-b.html
    └── ...
```

- `pages/` 目录下每个 HTML 文件为一个独立页面。
- 文件名建议使用简短、有意义的英文或拼音，便于识别。
- 根目录下的 `index.html` 为可选的导航页，用于展示仓库内所有页面的列表。

## 页面归档说明

- 所有页面均为静态 HTML，不包含动态内容或外部依赖。
- 页面内可能包含内联样式、脚本或图片（Base64 或相对路径）。
- 每个页面应尽量自包含，避免对外部资源的强依赖，以保证离线或迁移后仍可正常显示。
- 页面内容不保证长期更新，仅作为归档用途。

## 维护说明

- 本仓库由个人维护，不定期更新。
- 欢迎提交 Issue 或 Pull Request，但请遵守以下原则：
  - 不添加与现有页面无关的商业或推广内容。
  - 新增页面应保持独立、简洁，不破坏现有结构。
  - 修改现有页面时请说明原因。
- 仓库不提供任何形式的服务承诺或技术支持。

## 使用方式

1. 克隆或下载本仓库。
2. 直接双击打开 `pages/` 下的 HTML 文件，即可在浏览器中查看。
3. 也可将整个仓库部署到任何静态文件服务器或托管平台。
